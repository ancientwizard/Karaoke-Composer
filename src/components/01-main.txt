NOTES:
 Playback Counter HH::MM.SS
 Playback position is marked by a virtical blue line that intersects the whole hight of the RIGHT-SIDE content
   starting at the top of the narrow strip and ending at the top of the scroll bar

 BTN: Play / Pause "obvious"
 BTN: Recalculate <hover> == "Recalculate the preview to reflect the current timeline layout"
 BTN: Add Text...
      seems to add an empty "text" object to the first row but then it can be dragged to another row
 BTN: Add BMP...
      opens a file chooser and places it in the first row. but then can be dragged to another row.
 BTN: Set Wave...
      opens a file chooser to select the audio file for the project
 BTN: Export CDG
      opens file chooser to set where to render .cdg
 CHOOSE: <Sample-Rate>
   22,050 Hz
   32,000 Hz
   44,100 Hz <default>
   48,000 Hz
